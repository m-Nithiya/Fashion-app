Fashion Store App

A responsive Fashion Store mobile application developed using Ionic Angular. The application provides a modern shopping experience with features like product browsing, wishlist, shopping cart, checkout, and user profile management.


 Project Overview

The Fashion Store App is an e-commerce mobile application designed to help users browse fashion products for Men, Women, and Kids. Users can search products, add items to their wishlist or shopping cart, and complete the checkout process through an intuitive and responsive user interface.


 Features

- User Login
- Home Page
- Product Categories (Men, Women, Kids)
- Product Search
- Wishlist
- Shopping Cart
- Checkout Page
- Order Success Page
- User Profile
- Responsive Mobile UI

 Installation

Clone the Repository

```bash
git clone https://github.com/m-Nithiya/Fashion-app.git
```

Go to Project Folder

```bash
cd Fashion-app
```

Install Dependencies

```bash
npm install
```

Run the Application

```bash
ionic serve
```
